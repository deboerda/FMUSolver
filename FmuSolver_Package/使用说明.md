# FMU 仿真求解器部署指南

本文件夹包含了运行 FMU 仿真所需的所有核心组件，支持在无 Python 环境的电脑上直接运行。

## 目录文件说明
- **FmuControllerGUI.exe**: 主程序（图形化界面 + 仿真引擎）。
- **FMU/**: 存放模型文件（.fmu）。
- **Orbit_Pre.txt**: 当前模型所需的外部初始化数据文件。
- **MWSolver.dll / msvcr100.dll 等**: 32位仿真底层的核心运行库。

## 使用步骤
1. 将 **FMUSOLVER** 文件夹整体拷贝到目标机器。
2. 双击运行 `FmuControllerGUI.exe`。
3. 在界面中配置或浏览选择 `FMU/MTLibrary_System_EPS_PD_Orbit_Pre.fmu`。
4. 点击 **▶ 启动仿真**。
5. 仿真运行过程中，可以在界面中实时修改参数并点击 **♻ 应用参数** 进行热更新。

## 注意事项
- 本程序集成了 32 位运行环境，请确保在 Windows 系统上运行。
- 如果更换了新的 FMU 模型，请确保其依赖的外部 .txt 或 .dll 文件也放在本 EXE 同级目录下，程序会自动进行环境修复和路径注入。
