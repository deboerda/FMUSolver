# FmuSolver 源代码编译与打包指南

本文件夹包含了 FmuSolver 及其 Sidecar 架构的所有核心源代码，用于后续的代码审计与二次开发备份。

## 1. 环境准备
- **主环境**: 建议使用 Python 64-bit (推荐 3.10+)。
- **32位网桥环境**: 必须安装 Python 32-bit (例如安装在 `C:\Py32\python.exe`) 才能编译 32 位求解器。
- **必要库**: 
  ```bash
  pip install fmpy protobuf pyinstaller lxml numpy
  ```

## 2. 编译打包指令 (PyInstaller)

### 打包 64位 主程序 (FmuSolver.exe)
在 64 位 Python 环境下执行：
```bash
python -m PyInstaller --onefile --noconsole --name FmuSolver --icon satellite_icon_149794.ico FmuLauncherUniversal.py
```

### 打包 32位 求解引擎 (FmuWorker32.exe)
在 32 位 Python 环境下执行：
```bash
C:\Py32\python.exe -m PyInstaller --onefile --name FmuWorker32 --icon satellite_icon_149794.ico worker_entry.py
```

### 打包 64位 求解引擎 (FmuWorker64.exe)
在 64 位 Python 环境下执行：
```bash
python -m PyInstaller --onefile --name FmuWorker64 --icon satellite_icon_149794.ico worker_entry.py
```

## 3. 文件功能说明
- **FmuLauncherUniversal.py**: GUI 主程序，负责架构识别与进程分发（Sidecar 核心）。
- **fmu_player.py**: 仿真的核心库，处理 FMI 接口调用、UDP 数据打包、热更新接收。
- **worker_entry.py**: 求解器的精简入口，用于编译成 Worker.exe。
- **fmu_handler.py**: 处理 FMU 内部二进制文件的架构检测。
- **ddm.proto**: Protobuf 协议定义文件。
- **ddm_pb2.py**: 已编译的 Python Protobuf 接口类。
- **satellite_icon_149794.ico**: 应用卫星图标。

## 4. 依赖文件
- 运行目录下需放置 `MWSolver.dll` 及 `Orbit_Pre.txt` 才能完成特定模型的加载测试。
